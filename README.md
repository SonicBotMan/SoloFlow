> AI-Powered One-Person Video Production Company - 让一个人也能拥有完整的视频创作团队

## 📌 项目简介

**AI一人公司** 是一个基于AI Agent的智能视频创作平台，旨在让单个用户（"老板"）拥有一个完整的AI员工团队来完成视频创作工作。

用户只需要像老板一样下达指令，AI助理会协调各个AI员工完成从创意策划到视频发布的所有工作。

## 👥 AI员工团队

| 花名 | 角色 | 职责 | 核心能力 |

| 小助 | 老板助理 | 主调度、唯一对话入口 | 意图理解、任务分发、偏好学习 |
| 小点 | 点子王 | 热点分析、创意构思 | 趋势分析、创意推荐 |

| 小剪 | 剪辑师 | 视频剪辑、后期制作 | 视频剪辑、特效调色 |
| 小营 | 营销专家 | 营销策略、推广方案 | 平台分析、策略制定 |
| 小发 | 发布专家 | 多平台发布、数据追踪 | 自动发布、数据追踪 |

## ✨ 核心特性

### 1. 唯一对话入口
只有老板助理直接与用户对话，其他AI员工不直接面向用户，确保交互简洁。

### 2. 任务闭环
每个任务都有完整的流程：需求理解 → 任务分配 → 执行 → 汇报 → 确认 → 反馈

### 3. 偏好学习系统
- 老板的每次反馈都会被自动提取为偏好标签
- 员工会记住老板的喜好并在后续工作中自动应用
- 支持查看员工对老板偏好的理解程度

### 4. 掌控感设计
- **项目中心**: 查看所有项目进度
- **员工中心**: 查看员工状态、历史工作、偏好理解
- **偏好面板**: 查看和管理自己的偏好设置

## 🚀 快速开始

### 方式一：Docker Compose（推荐）

```bash
# 1. 克隆项目
git clone https://github.com/SonicBotMan/ai-one-person-company.git
cd ai-one-person-company

# 2. 配置 API Key
cp .env.example .env
# 编辑 .env 文件，填入你的 OPENAI_API_KEY

# 3. 一键启动
docker compose up

# 打开 http://localhost:8000
```

### 方式二：本地运行

```bash
# 1. 克隆项目
git clone https://github.com/SonicBotMan/ai-one-person-company.git
cd ai-one-person-company

# 2. 安装依赖
pip install -r requirements.txt

# 3. 配置环境变量
export OPENAI_API_KEY=sk-xxx

# 4. 启动
python main.py

# 或使用 CLI 模式
python main.py --cli
```
