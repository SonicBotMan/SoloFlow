# 国内模型配置指南

SoloFlow 支持接入国内大模型，降低使用成本，提升响应速度。

---

## 📋 支持的模型

| 提供商 | 模型 | 特点 | 费用 |
|--------|------|------|------|
| **MiniMax** | abab6.5-chat | 海螺AI，中文优秀 | 0.015元/千tokens |
| **智谱AI** | glm-4 | GLM-4，能力强 | 0.1元/千tokens |
| **Moonshot** | moonshot-v1-8k | Kimi，长文本 | 0.012元/千tokens |
| **百度** | ernie-bot-4 | 文心一言 | 0.12元/千tokens |
| **阿里** | qwen-turbo | 通义千问 | 0.008元/千tokens |

---

## 🚀 快速配置

### 方式一：环境变量（推荐）

```bash
# MiniMax
export MINIMAX_API_KEY="your_minimax_key"
export MINIMAX_GROUP_ID="your_group_id"

# 智谱AI
export ZHIPU_API_KEY="your_zhipu_key"

# Moonshot
export MOONSHOT_API_KEY="your_moonshot_key"

# 百度
export BAIDU_API_KEY="your_baidu_key"
export BAIDU_SECRET_KEY="your_secret_key"

# 阿里
export DASHSCOPE_API_KEY="your_dashscope_key"
```

### 方式二：.env 文件

```bash
# .env 文件
MINIMAX_API_KEY=your_minimax_key
ZHIPU_API_KEY=your_zhipu_key
MOONSHOT_API_KEY=your_moonshot_key
```

## 🔧 代码中使用

### 使用国内模型客户端

```python
from soloflow.models import DomesticModelClient

# MiniMax
client = DomesticModelClient("minimax")
response = await client.chat(
    messages=[{"role": "user", "content": "你好"}],
    temperature=0.7
)

# 智谱AI
client = DomesticModelClient("zhipu")
response = await client.chat(messages, temperature=0.7)

# Moonshot
client = DomesticModelClient("moonshot")
response = await client.chat(messages, temperature=0.7)
```

### 在 Agent 配置中使用

```yaml
# agents/assistant.yaml
name: assistant
alias: 小助
model: minimax  # 改为国内模型
temperature: 0.3
```

## 🔑 API Key 获取

### MiniMax（海螺AI）

1. 访问：https://www.minimaxi.com
2. 注册账号
3. 创建应用
4. 获取 API Key 和 Group ID

### 智谱AI（GLM）

1. 访问：https://open.bigmodel.cn
2. 注册账号
3. 创建 API Key
4. 获取 Key

### Moonshot（Kimi）

1. 访问：https://platform.moonshot.cn
2. 注册账号
3. 创建 API Key

### 百度文心一言

1. 访问：https://console.bce.baidu.com/qianfan
2. 创建应用
3. 获取 API Key 和 Secret Key

### 阿里通义千问

1. 访问：https://dashscope.console.aliyun.com
2. 创建 API Key

---

## ⚠️ 注意事项

1. **限流**：各平台都有调用频率限制，注意控制
2. **费用**：设置费用告警，避免超支
3. **备份**：建议配置多个模型，互为备份
4. **测试**：上线前务必测试模型能力

## 🔄 模型切换策略

```python
from soloflow.models import chat_with_model

# 优先使用国内模型
try:
    response = await chat_with_model(
        provider="minimax",
        messages=messages,
        temperature=0.7
    )
except Exception:
    # 失败时切换到 GPT-4
    response = await chat_with_model(
        provider="openai",
        messages=messages,
        temperature=0.7
    )
```
